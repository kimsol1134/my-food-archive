# Android 온라인 실습 자료

이 자료는 Android 앱 개발과 Google Play 배포가 처음인 독자를 위한 온라인 실습 가이드입니다. 사용 중인 컴퓨터가 Windows인지 macOS인지와 관계없이, 아래에서 현재 읽고 있는 장을 선택해 순서대로 진행하면 됩니다.

## 지금 시작할 곳

현재 읽는 장에 맞는 자료 하나만 열면 됩니다.

- 15~16장: **[Android 앱을 Google Play 내부 테스트에 올리기](google-play-internal-test-guide.md)**
- 17~18장: **[비공개 테스트에서 Google Play 공개까지](release-roadmap.md)**

Google Play 배포가 처음이라면 15~16장 자료부터 시작하세요. Play Console, AAB, 테스트 트랙, 참여 링크를 1절에서 먼저 설명합니다.

## 책과 온라인 자료를 함께 읽는 순서

Google Play의 계정 확인과 심사는 바로 끝나지 않을 수 있습니다. 아래 표에서 현재 상태에 맞는 행만 진행합니다.

| 책 | 온라인 자료를 여는 시점과 범위 | 책으로 돌아갈 곳 |
|---|---|---|
| 15장 | 내부 테스트 가이드 1~11절. 9.1절은 건너뜁니다. 계정 확인을 기다리면 5절부터, 테스트 버전 반영을 기다리면 9절부터 다시 시작합니다. | 책 15.5절 |
| 16장 | 책에서 앱 수정까지 먼저 진행한 뒤, 16.4절의 Android 안내에서 가이드 9.1절만 열어 재업로드와 업데이트 확인 | 책 16.5절 |
| 17장 첫 방문 | 공개 로드맵 1~6절 — 제출 정보와 릴리스 설정 점검 후 비공개 테스트 시작 | 책 17.5절 |
| 14일 조건 충족 후 | Play Console에 신청 가능 상태가 표시되면 공개 로드맵 7절 | 프로덕션 접근 결과 기다리기 |
| 책 18장·승인 대기 중 | 공개 로드맵 9절에서 기존 비공개 테스터가 열 수 있는 링크 확인 | 책 18.3절 |
| 책 18장·프로덕션 접근 승인 완료 | 공개 로드맵 8~9절 — 프로덕션 출시와 정식 링크 확인 | 책 18.3절 |

## 예상 시간

- 개발자 계정 본인 확인: 바로 끝나지 않고 며칠 걸릴 수 있음
- 첫 내부 테스트 반영: 수분에서 길게는 48시간
- 새 개인 계정의 비공개 테스트: 최소 12명·14일
- 프로덕션 접근과 앱 심사: 추가 검토 시간 필요

15~16장에서는 내부 테스트 설치를 목표로 합니다. 17장에서 비공개 테스트를 시작하고, 14일 조건과 프로덕션 접근 승인이 끝나지 않았다면 기존 비공개 테스터와 링크를 확인하며 18.3~18.4절을 진행합니다. 새 사람에게 보낼 때는 먼저 그 사람의 Google 계정을 비공개 테스터 목록에 추가합니다.

## 문제가 생겼을 때

Google Play Console의 화면 이름과 요구사항은 자주 바뀝니다. 이 자료의 텍스트와 실제 화면이 다르면 원본 화면을 캡처해 클로드 코드에게 보여 주세요. 계정 이메일과 다른 앱 이름은 그대로 두고, 주소·결제 정보·비밀번호·인증서 지문이 보이는 화면은 캡처하지 않습니다.
