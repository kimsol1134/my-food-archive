## 앱 안의 개인정보처리방침

![앱 안에서 연 개인정보처리방침](../book-screenshots/android-app/a-06-privacy-policy.png)

Google Play에 제출할 때는 홈, 자동 입력, 저장 결과, 검색 결과 중 앱의 핵심 흐름이 잘 이어지는 화면을 2장 이상 선택합니다. 화면을 자르거나 문구를 합성하지 않고 Android폰에서 촬영한 원본을 사용합니다.
