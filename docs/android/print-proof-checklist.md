## 앱 화면

Android 앱 화면 6장은 모두 1080×2400px입니다. 300dpi 기준 약 91×203mm까지 원본 픽셀을 유지합니다.

- [ ] 같은 음식 사진과 식당명이 A-2~A-5에서 자연스럽게 이어짐
- [ ] 상태바의 시간·알림이 독자 시선을 방해하지 않음
- [ ] Gemini 결과에서 메뉴와 카테고리만 자동 입력되고 식당명은 비어 있음
- [ ] 개인정보처리방침 화면에서 AI 사진 전송과 App Check 안내를 읽을 수 있음

## Google Play Console 화면

현재 캡처는 500~1068px 폭의 온라인 자료용 타이트 크롭입니다. 종이책에서 다음 조건 중 하나라도 해당하면 Arc 브라우저 원본 해상도로 다시 캡처합니다.

- 편집면에서 120mm보다 크게 사용
- 본문 글자나 버튼명이 100% 확대에서 흐림
- 캡처 일부를 다시 확대해 사용하는 구성

재캡처할 때는 브라우저 배율 100%, 전체 콘텐츠 폭 1440px 이상을 기준으로 하고 이메일, 실명, 주소, 전화번호, 인증서 지문, 전체 테스트 참여 링크를 제외합니다.
