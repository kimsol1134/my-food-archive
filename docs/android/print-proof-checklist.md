## 앱 화면

Android 앱 화면 6장은 모두 1080×2400px입니다. 300dpi 기준 약 91×203mm까지 원본 픽셀을 유지합니다.

2026년 7월 13일 인쇄 교정용 PDF에 폭 75mm로 배치하고 150dpi PNG로 다시 렌더링해 확인했습니다. 제목, 입력값, 저장 결과, 검색어와 개인정보처리방침 본문의 핵심 문장이 판독됐습니다. 최종 지면에서도 폭 75mm 이상을 사용합니다.

- [x] 같은 음식 사진과 식당명이 A-2~A-5에서 자연스럽게 이어짐
- [x] 상태바의 시간·알림이 독자 시선을 방해하지 않음
- [x] Gemini 결과에서 메뉴와 카테고리만 자동 입력되고 식당명은 비어 있음
- [x] 개인정보처리방침 화면에서 AI 사진 전송과 App Check 안내를 읽을 수 있음

## Google Play Console 화면

책과 온라인 자료에는 촬영 후 편집하지 않은 원본 캡처만 사용합니다. 용도별 공개 범위는 구분합니다.

- 자르기, 리샘플링, 합성, 블러, 가림, 화살표, 박스, 번호 추가 금지
- 전체 창을 원본 해상도로 캡처하되, 최종 PDF에서 핵심 버튼과 상태 문장이 읽히지 않으면 같은 화면을 브라우저 배율 125%·150%·175% 중 하나로 다시 촬영
- 배율을 높여 다시 찍은 화면도 촬영 뒤 자르거나 확대하지 않은 원본 파일로 제출
- 출판사 보관·책 배치용 원본은 계정 이메일, 다른 앱 이름, 주소, 전화번호와 결제 정보가 보여도 그대로 제출하고 편집자가 최종 지면에서 처리
- 온라인 공개용 원본은 이메일과 다른 앱 이름을 유지할 수 있지만, 주소·전화번호·결제 정보가 화면에 나오지 않는 상태에서 촬영
- 클로드 코드 전달용 화면에는 이메일·주소·전화번호·결제 정보를 포함하지 않음
- 비밀번호, 인증번호, 인증서 지문 전체와 전체 테스트 참여 링크는 모든 용도에서 촬영하지 않음

Play Console 20장은 2026년 7월 13일 원본 캡처로 교체했습니다. P-15~P-20은 현재 실제 Play Console 화면에서 추가로 촬영한 JPEG 원본입니다.

### 인쇄 크기 판독 결과

2026년 7월 13일 [`build_android_print_proof.py`](../../scripts/build_android_print_proof.py)로 앱 화면 6장과 Play Console 화면 20장을 한 장씩 배치한 26쪽 교정용 PDF를 만들었습니다. Play Console 화면은 폭 130mm, 앱 화면은 폭 75mm로 배치한 뒤 150dpi PNG로 다시 렌더링해 모든 페이지를 확인했습니다.

| 원본 | 확인한 배치 폭 | 결과 |
|---|---:|---|
| Android 앱 화면 A-1~A-6 | 75mm | 화면 제목, 입력값, 저장·검색 결과와 개인정보처리방침의 핵심 문장 판독 가능 |
| Play Console 화면 P-1~P-20 | 130mm | 본문에서 지목하는 메뉴, 버튼, 트랙 이름과 상태 문장 판독 가능 |

교정용 PDF는 개인정보가 포함된 원본의 중간 산출물이므로 저장소에 올리지 않습니다. 필요할 때 프로젝트 루트에서 다음 명령으로 다시 만듭니다.

```bash
python3 scripts/build_android_print_proof.py
```

최종 편집 PDF에서 위 최소 폭보다 작게 배치한 화면은 이 판독 결과의 적용 대상이 아닙니다.

## 최종 PDF 확인

- [ ] QR이 25mm 이상이며 네 변의 흰 여백이 유지됨
- [ ] QR을 iPhone과 Android 기본 카메라에서 각각 스캔함
- [ ] 링크가 로그인 없이 열림
- [ ] PDF에 배치한 이미지 파일이 촬영 원본이며 별도 편집본이 아님
- [ ] Play Console 화면은 폭 130mm 이상, Android 앱 화면은 폭 75mm 이상으로 배치됨
- [ ] 최종 PDF를 실제 인쇄 크기 100%로 보았을 때 본문에서 설명하는 버튼과 상태 문장을 읽을 수 있음
- [ ] 글자가 작으면 지면에서 이미지를 확대 편집하지 않고, 더 높은 브라우저 배율로 다시 촬영한 원본으로 교체함
- [ ] 최종 PDF에서 주소, 전화번호와 결제 정보가 가려짐
- [ ] 온라인 공개 화면 원본에 주소, 전화번호와 결제 정보가 없음
- [ ] 촬영 원본과 최종 PDF에 비밀번호, 인증번호, 인증서 지문과 전체 참여 링크가 포함되지 않음
- [ ] 온라인 자료의 장별 복귀 지점과 최종 DOCX가 일치함
